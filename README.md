こちらのリポジトリは「Web開発者のための［入門］Cloudflare Workers」で一部抜粋して掲載したコードの完全版です。

## 使い方

1. このリポジトリをクローンします。`git clone invitation-to-cfw-durable-objects-sample.git`
2. `counter-do`もしくは`chat-do`へ移動します。`cd counter-do`,`cd chat-do`
3. `npm install`で依存関係をインストールします。
4. `npm run cf-typegen`でTypeScriptの型定義を生成します。
5. `npm run dev`でWrangler CLIから開発用サーバーを起動します。
